2. Überblick – Welche Daten werden verarbeitet?

Im Rahmen der Nutzung der App können folgende Daten verarbeitet werden:

  • Inhaltsdaten (z. B. Einträge/Informationen, die Sie in der App erstellen oder speichern)
  • Technische Daten (z. B. interne IDs, Zeitstempel, Fehler-/Sicherheitsinformationen, die für den Betrieb notwendig sind)
  • Medien (z. B. Bilder), sofern Sie Funktionen nutzen, die Medien aufnehmen/auswählen

Hinweis: Die App ist so ausgelegt, dass nur die für die Funktionen notwendigen Daten verarbeitet werden.

3. Nutzung der Kamera (Berechtigung: android.permission.CAMERA)

Die App kann Zugriff auf die Kamera benötigen, um bestimmte Funktionen bereitzustellen (z. B. Foto aufnehmen oder Inhalte erfassen). Die Kamera wird nur verwendet, wenn Sie diese Funktion aktiv starten.

Aufgenommene Inhalte werden – abhängig von der von Ihnen genutzten Funktion – lokal auf dem Gerät und/oder im Backend gespeichert.
